This project shows data analysis using Excel. Taking raw Sample dataset of Food Store from kaggle . Dashboard analyzes revenue performance , providing insights into:-
    -Product categories, payment preferences, revenue trends, store growth over time.
    This is just a simple part of learning project with analytics -driven.

    OBJECTIVE:
    -Identify top- selling Type or category
    -Track total revenue over time
    -Understand seasonal or monthly variations 
    -comparison of segments
    -data backed insights

    DATASET:-
    -Type:Food store
    -Contains: Order ID and Date, Item type, Price,	quantity, transaction_amount,transaction_type,time_of_sale
    -Source: Kaggle

    The problem statements and Insights are given inside the excel file.
